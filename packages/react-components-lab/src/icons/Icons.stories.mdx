# DHI Icons

These icons are customized Material Design icons, with new icons specific to DHI. The icons are designed to be simple, modern, friendly, and sometimes quirky.

## Usage

In order to use the package, please install it locally in your project dependencies.

**Note**: In order for the `@dhi` GitHub packages registry to be found, please follow the [instructions]('https://github.com/DHI/react-components/tree/master/packages/icons#installing-the-package').
```
yarn add @dhi/icons
// or
npm install @dhi/icons
```
