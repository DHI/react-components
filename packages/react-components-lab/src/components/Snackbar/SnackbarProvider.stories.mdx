# Snackbar

`Snackbar` is used to display a message, known as notification / toast. For the component to work, you have to use the `SnackbarProvider` together with the `useSnackbar` hook.

<Preview>
  <Story story={stories.Default} />
</Preview>

## How to use

1. Wrap your main app within `ThemeProvider` and `SnackbarProvider`. If you just passing nothing to
`SnackbarProvider` props, then it will use default values and this can be overridden through `useSnackbar` hooks.
Just info, passing `Props` values at this level means the values will affect globally.

### Component with no passing Props values

```tsx
import { SnackbarProvider, ThemeProvider } from '@dhi/react-components-lab/SnackbarProvider';

<ThemeProvider>
  <SnackbarProvider>
    <App />
  </SnackbarProvider>
</ThemeProvider>
```

### Component with passing Props values

```tsx
import { SnackbarProvider, ThemeProvider } from '@dhi/react-components-lab/SnackbarProvider';

<ThemeProvider>
  <SnackbarProvider autoHideDuration={5000}>
    <App />
  </SnackbarProvider>
</ThemeProvider>
```

2. Use `useSnackbar` hooks. This hooks used to display a message and can be used to pass the `Props` values too.
Passing `Props` values at this level only affect to individual `Snackbar`.

### Hooks with no passing Props values

```tsx
import React from "react";
import { useSnackbar } from "@dhi/react-components-lab";
import { TextField, Button } from "@material-ui/core";

const App = () => {
  const snackbar = useSnackbar();

  const showMessage = () => {
    snackbar.showMessage('Hello world!');
  };

  return <Button onClick={showMessage}>Show</Button>;
};

export default App;
```

### Hooks with passing Props values

```tsx
import React from "react";
import { useSnackbar } from "@dhi/react-components-lab";
import { TextField, Button } from "@material-ui/core";

const App = () => {
  const snackbar = useSnackbar();

  const showMessage = () => {
    snackbar.showMessage('Hello world!', { severity: 'success' });
  };

  return <Button onClick={showMessage}>Show</Button>;
};

export default App;
```
